tourne_de → tourne les étoiles
rotation : valeur de rotation

dessine_etoiles → dessine toutes les étoiles
nombre : nombre d'étoile
taille : taille des étoies
type : type d'étoile (options possibles : 'rond', 'carré', 'dessin')
